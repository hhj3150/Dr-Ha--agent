## 1. 업로드 문서 4건 요약

### ① 별표1 — 농업경영회생자금 「농업경영위기(예시)」
경영위기로 인정되는 사유:
- **재해**(농어업재해대책법): 가뭄·홍수·호우·해일·태풍·강풍·이상저온·우박·서리·조수·대설·한파·폭염·병해충·일조량 부족·유해야생동물 등
- **그 밖의 사유**:
  - 방화 아닌 전기누전·낙뢰 등 화재
  - 보증인이 농업용 자금 대위변제(할 경우)
  - 이상기후로 전년比 생산량 감소 → 농업소득 감소
  - **원자재(비료·농약·사료·유류·시설자재) 가격 15%↑ 상승** 또는 **농산물 시장가격 15%↓ 하락** (청년농 10%)
  - 수출 불가/환율 급락(15%↓, 청년농 10%)
  - 1년 이상 무연체 농업인이 최근 **3개월 이상 연체** 발생
  - 경영회생지원 농지매입사업 대상자 선정(농어촌공사 확인서)
  - 본인/인력 **감염병 확진·격리** → 정상 영농 불가
  - 감염병 영향(납품취소·급식중단 등)으로 생산·판매·매출 **15%↓**(청년농 10%)
  - 병해충 피해로 매출 **15%↓**
- 청년농 기준: **만 40세 미만** 농업 종사자

### ② 별표2 — 품목별 준전업농 규모 기준(예시)
축산 기준(자격 판단 핵심):
| 품목 | 준전업농 규모 | 비고 |
|---|---|---|
| 한우 | 25두 이상 | 번식우는 33두 이상 |
| **젖소** | **25두 이상** | **착유마리수 기준** |
| 돼지 | 500두 이상 | 모돈 기준 50두 이상 |
| 닭·오리 | 15,000수 이상 | |
| 흑염소 | 250두 이상 | |
| 꽃사슴 | 65두 / 레드미어 100두 / 엘크 | 수컷 기준 |
- **복합영농**: 농산물 판매금액 **연 2,000만원 이상**도 준전업농 인정
- **임차농**: 임차농지도 100% 경작지 포함(임대차계약서·토지대장 등 증빙)
- ※ 준전업농 = **전업농 규모의 1/2 이상**

### ③ 농업자금이차보전 사업개요(2026, 13p)
- **예산 729,661백만원(약 7,296억), 운용자금 8.9조원**
- 구조: 농협 등 금융기관 자체자금으로 저리 대출 → 정부가 이자차액 보전
- 근거: 농어업인 부채경감 특별조치법, 농업·농촌식품산업기본법, 농림사업정책자금 이차보전규정
- **금리(변동)**: 이차보전 기준금리에서 **농업(법)인 2.0%p 차감 / 일반기업 1.0%p 차감**
- 신청: 농협은행·지역 농·축협, **연중(자금소진 전까지)**
- 재해 간접지원: 농가단위 피해율 30~50% → 상환연기·이자감면 1년 / 50%↑ → 2년
- **이차보전 대상사업(우리 관련)**: 농축산경영자금, 농업종합자금, 농업경영회생자금,
  **축산경영종합자금, 축사시설현대화, 축산물도축가공업체지원, 사료구매자금,
  식품소재업계 경영안정자금(제분·원당·올리브유 등), 농산업수출업체 시설현대화·운영지원** 등

### ④ 농업경영회생자금(이차보전 세부)
- 목적: 재해 등 **일시적 경영위기** 농업인에게 저리 정책자금
- 근거: 농어업인 부채경감 특별조치법 제5조의2
- 대상: **준전업농(전업농 1/2 이상) 또는 농업용 부채 보유 농업인** (영농조합법인·농업회사법인 포함)
- **★지원조건: 고정 1%, 5년거치 7년 분할상환(원금균등), 개인 20억 / 법인 30억 이내**
  - 융자 90% 이내, **자부담 10% 이상** 농업인 우선지원
- '26년 운용자금(융자): **30,000백만원(300억)**
- 지원대상 자금: 5년 내 상환도래 농업용 대출 원리금, 협동조합 경제사업 연체채무(사료대 등),
  피해복구 시설·개보수 자금, 대위변제 농업용 자금, 품목별 1회전 운전자금
- 신청: 농협은행 시군지부/지역 농·축협, 연중
- **선정 절차**: 대출취급기관 경영평가단 사전평가 → 정밀경영평가 → 경영평가위원회 결정
  - **4억원 초과**: 농협은행 중앙본부 경영평가위원회 심사
  - **4억원 이하**: 대출취급기관 평가(사전+정밀)로 지원
  - 평가결과: 정상 / 회생가능(자금지원) / 회생불능(시설인수자 있으면 인수자금)
- **지원제외**: 농협·산림·수협 상근 임직원, 공무원·교직원·공공기관 정규직, 비농업용 부동산(1주택 제외)·골프회원권 보유자; 본인·배우자 모두 농업경영정보 미등록(둘 중 1명 등록 시 가능)
