# 자료실 (materials) — 하실장 원자료 보관소

> 회장님이 주신 원자료(문서·계획서·계약서·데이터)를 여기에 보관합니다.
> 보관된 자료는 **하실장(빌더)이 직접 읽어** 분석·산출물 작성에 사용합니다.
> 이 저장소는 **비공개(private)** 이며, 외부에 공개되지 않습니다.

## 작동 방식 (회장님 → 하실장 → GitHub → 사용)
1. **회장님이 자료를 줌** — ① 이 채팅에 파일 업로드 ② 복사-붙여넣기 ③ 구글드라이브/Notion 링크
2. **하실장이 GitHub에 저장** — 원자료는 `materials/`, AI가 항상 기억해야 할 핵심은 `context/`
3. **자동 사용** — 빌더 하실장·웹앱은 저장소를 직접 읽음 / 프로젝트 하실장은 지식 묶음 재업로드로 동기화
4. **인코딩 깨진 한글파일**(.txt mojibake)은 하실장이 UTF-8로 복구 후 보관

## 보관 규칙
- 민감자료(사업자번호·계약·내부전략)는 **이 private 저장소에만** 보관. 외부 산출물에는 기능·전략 의미만 사용.
- 기밀(Dr. Ha Liquid 원천·메탄/모넨신/풀빅산 전략·내부 배합·비공개 파트너)은 외부 노출 금지.
