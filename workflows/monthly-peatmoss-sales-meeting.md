# Monthly Peat Moss Sales Meeting Workflow

Use to run the monthly peat-moss sales review. Owner: D2O Peat Moss Commercial Agent.

## Agenda
1. Sales scoreboard vs target (volume, revenue, by segment/region/dealer)
2. Pipeline review (new proposals, trials, closes)
3. Dealer/distributor performance
4. 농협/축협 & government-linkage progress
5. Inventory / logistics / import schedule status
6. Repeat-purchase & CRM actions
7. Issues & blockers
8. Next-month targets & action assignment

## Output (use monthly-sales-report-template)
- Filled scoreboard
- Pipeline table (stage, value, next step)
- Dealer performance table
- Inventory/logistics status
- Next-month targets & owner actions

## Rule
Numbers are targets unless actuals confirmed. Link government-related performance to
the Government R&D Secretary Agent when relevant.
