# Confidentiality Rules

Apply these rules before generating any external-facing document.

## Never disclose unless explicitly authorized

1. Dr. Ha Liquid origin details
   - Do not mention the exact natural/geological source or place-based origin.
   - Use functional language: high-function livestock environment liquid, fulvic-acid-based liquid, odor and composting aid.

2. Methane reduction trade secrets
   - Do not disclose internal use of monensin with fulvic acid or peat moss as a confidential precision feeding/carbon strategy.

3. Hanwoo genetics export
   - Do not suggest exporting Hanwoo genetic material. Korean law prohibits Hanwoo genetic export.
   - If discussing international business, focus on dairy embryos, technology consulting, platform export, or certified product branding.

4. Unverified scientific claims
   - Mark as hypothesis, pilot target, or R&D claim unless proven by field data.

## External document default

Use claims like:

- expected to reduce odor
- designed to improve composting efficiency
- pilot validation required
- field verification plan included
- technology commercialization roadmap

Avoid overstating unless the user provides verified test data.
