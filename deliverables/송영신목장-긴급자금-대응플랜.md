## 2. 1순위 솔루션 — 농업경영회생자금 대환 (개인/송영신목장)

**무엇**: 사료정책자금 3억을 농업경영회생자금으로 **전환(대환)**
**효과**:
- 금리 **고정 1%**, **5년 거치** 7년 분할상환(원금균등)
- → **10월 상환부담 즉시 해소**, 거치기간 5년간 **원금상환 유예** = 유통채널 정착 runway 확보
- 융자 90% 이내 + 자부담 10% 이상 농업인 우선

**자격 충족 논리**:
1. **농업용 부채 보유 농업인** = 사료정책자금 3억(농업용) 보유 → **준전업농 아니어도 기본 자격 ✓**
2. (병행) **준전업농** = 젖소 **착유 25두 이상**이면 추가 충족 → 착유두수 확인 필요
3. **경영위기 사유** = 별표1 "기타 경영평가위원회가 일시적 경영위기로 인정하는 경우" + **사료가격 상승분**
4. **지원대상 자금** = "5년 이내 상환도래 농업용 대출금 원리금" → 10월 도래분 정확히 해당

**절차상 이점**: 신청액 **3억 < 4억** → **경영평가위원회 생략**, 대출취급기관(지역 농·축협/농협은행) 사전+정밀평가로 처리 → 상대적으로 신속

**★ 승부처 = 자구계획서(회생 스토리)**
- 10년 건실 영위 실적 → 고부가(저지 A2·목장형 유가공) 전환 투자 → 일시적 현금 갭
- **회생계획**: 유통채널(shop.a2jerseymilk.com, B2B, 정기구독) 구축으로 **18개월 내 매출 정상화** 목표·수치 제시
- 위기사유로 **사료가 상승** 객관 증빙 첨부

| 우선 | 옵션 | 내용 | 비고 |

| 즉시 | **사료정책자금 상환연기(유예) 신청** | 기존 자금 자체를 경영위기 사유로 상환연기 → 10월 도래분 브릿지 | 회생자금 실행 전 시간벌기 |
