# shop.a2jerseymilk.com — E-Commerce Launch Checklist Workflow

Pre-launch readiness for **shop.a2jerseymilk.com**. Owner: e-Commerce Agent (with
Brand Marketing for content, Sales for offers). Built from the 6 growth goals plus
premium D2C best practice.

> Live-audit note: the shop currently returns HTTP 403 to external automated
> checks, so this is a best-practice checklist. Confirm the builder (Cafe24 /
> Imweb / Shopify, etc.) and provide a screenshot to convert this into a
> platform-specific audit.

## 1. Foundation
- [ ] Domain, SSL/HTTPS, mobile-responsive verified
- [ ] Platform/builder confirmed; analytics installed (GA4 + pixel)
- [ ] Business info: 사업자등록·통신판매업 신고, 청약철회/교환·반품 정책, 개인정보처리방침, 이용약관

## 2. Information architecture
- [ ] Main page strategy (hero, value props, social proof)
- [ ] Category structure (milk / yogurt / greek / kaymak / ice cream / milk tea / sets / subscription)
- [ ] Navigation & search

## 3. Product detail pages (PDP)
- [ ] Brand story + Soil-to-Soul section
- [ ] Nutrition (provided lab values), ingredients, storage, shelf-life
- [ ] Photos/video, options, price, delivery info (cold-chain)
- [ ] Reviews block; FAQ

## 4. Conversion & checkout
- [ ] Cart, guest + member checkout, multiple payments (card/Naver Pay/Kakao Pay)
- [ ] Coupons, first-order offer, free-shipping threshold
- [ ] Cold-chain shipping & delivery-day rules

## 5. Membership, subscription & retention
- [ ] Membership tiers & points
- [ ] Subscription (정기구독): cadence, pricing, skip/pause, manage
- [ ] CRM, email, KakaoTalk channel flows (welcome, abandoned cart, repurchase)

## 6. Trust & growth
- [ ] Review system live; UGC plan
- [ ] SEO basics (titles, meta, structured data, sitemap)
- [ ] Customer service: inquiry, KakaoTalk, return/refund flow

## Output
A pass/fail checklist with owner per item, blockers for open, and a prioritized fix list to reach open-ready.
