# SmaXtec 국내 공급/단가 모델 (v2)

작성: 하실장 · 담당: smaxtec-sensor-operations + ceo-strategy · 분류: Tier 1 / Phase 1
상태: 내부 초안 · 근거: `context/smaxtec-sales-structure-context.md`(기밀), `context/gyeonggi-2027-ai-livestock-platform-context.md`
> ⚠️ 원가·마진·지분구조는 내부 한정 — 대외 제안서엔 판매가·서비스 조건만 사용.

## 1. 목적
경기도 정책연구가 명시한 "SmaXtec 반추위 센서 + 본사 기술 제휴"의 국내 주체가
**하현제(이미 국내 독점 영업권·API 보유)** 임을 전제로, 시범사업 7,750두 공급 단가·구조를 정리한다.

## 2. 현재 구조 (이미 확보된 권리 — 신규 요청 아님)
- **국내 SmaXtec 위내센서 영업: 하현제 전담(2025~).** 국내 판매 **독점** + **SmaXtec API 국내 독점** 보유.
- 공급 체인: SmaXtec(본사) → **코리아제네틱스(수입, 대표 장익준)** → **하현제(국내 독점 영업)** → 목장.
  (기존 거래관계는 리얼팜 유지. 리얼팜: 대표 장익준 / 최대주주 하현제.)
- 즉 경기도 사업에 센서를 넣을 **국내 유일 합법 채널 = 하현제.** 이게 입찰의 결정적 명분.

## 3. 판매가 / 서비스 조건 (대외 사용 가능)
| 항목 | 금액 | 조건 |
|---|---|---|
| 통신중계기 설치 + 인프라 구축 | **목장당 500만원** | 1회성(농장 단위) |
| 센서 이용료 | **소 두당 월 12,000원** | **4년 유지조건**, 중간 고장 **무상 교체** |
| API/데이터 | 국내 독점 | CowTalk AI 분석과 결합 |

## 4. 내부 단가/마진 (CONFIDENTIAL — 대외 금지)
- 공급 원가: **두당 월 7,700원**(하현제 → 코리아제네틱스, 2026-06-03 기준·변동 가능). 두당 월 마진 **4,300원**(≈35.8%).
- 설치 용역비: 리얼팜 직원 설치 시 **리얼팜에 250만원/목장** 지급 → **설치 마진 250만원/목장**.

## 5. 7,750두(155농장) 적용 단위경제 (예시·48개월 가정)
- 설치(1회성): 매출 155×500만 = **7.75억** / 용역비 3.875억 / **설치 마진 ≈ 3.875억**
- 구독: 매출 12,000×7,750×48 = **약 44.6억** / 원가 약 28.6억 / **구독 마진 약 16.0억**
- **총 마진(예시) ≈ 19.9억** (구독 16.0 + 설치 3.875, 48개월 가정)
- ※ 시범 공식기간은 36개월/3단계 — 위는 표준 농가계약(4년) 기준 예시. 과금 시점·단계에 따라 조정.
- 핵심: **두당 월 구독이 4년 반복매출 엔진.** CowTalk AI 분석 가치가 12,000원 단가를 정당화.

## 5. 7,750두 공급·설치 운영 계획
- 농장: 31시군 × 5농장 = 155농장, 농장당 50두.
- 단계: 1단계(0~6M) 설치 집중 → 데이터 수신율 검증 → 2단계부터 구독 과금.
- 운영도구: sensor-installation-checklist · monthly-sensor-subscription-report · sensor-as-response-template.
- 인력: 권역별 설치기사 배정 + 수의 현장점검(고려동물병원·제네틱스 연계).

## 6. 리스크 & 대응
- 수입 단가(7,700원) 변동 → 코리아제네틱스와 공공사업용 단가·물량 사전 합의.
- 실시간성 한계(보고서 지적) → 배치+엣지 수집, Eco-BIT 버퍼링, 본사 API 협의.
- 환율·수입 일정 → 코리아제네틱스 재고 버퍼·물류 일정 관리(공급계획표).
- 회사 간 역할 명확화 → 영업(하현제)·수입(코리아제네틱스)·기존거래(리얼팜) 경계 문서화.
