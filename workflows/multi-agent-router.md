# Multi-Agent Router Workflow

Use this workflow when the user gives a broad idea.

## Step 1: Classify the idea

Choose one or more domains:

- CEO Strategy
- Eco-BIT Platform
- Genetics Biotech
- D2O Environment Carbon
- A2 Hay Milk Brand

## Step 2: Decide output type

Common output types:

- government proposal
- patent draft
- investor memo
- farmer manual
- product brochure
- technical specification
- business model
- implementation ticket
- presentation outline

## Step 3: Assign owner agent

- Strategy first when ambiguous.
- Specialist first when the task is narrow.
- Use two agents when a strategy needs technical detail.

## Step 4: Produce action output

Always end with:

- next document to create
- next field data needed
- next meeting target
- next implementation task
