# Claude Code Build Workflow

Use when converting an idea into implementation tasks for Claude Code or Codex.

## Required output

```text
Project:
Goal:
User roles:
Core features:
Data model:
API endpoints:
UI screens:
Files to create:
Files to modify:
Acceptance criteria:
Test cases:
Mock data:
Deployment notes:
```

## Rule

Never ask Claude Code to build everything at once. Split work into small tasks:

1. data model
2. mock data
3. API
4. UI screen
5. integration
6. tests
7. polish
