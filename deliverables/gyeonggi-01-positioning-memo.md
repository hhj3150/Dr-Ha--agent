# 경기도 AI 축산행정 플랫폼 — D2O 참여 포지셔닝 메모 (v1)

수신: 회장님(하현제) · 작성: 하실장 · 분류: Tier 1 / Phase 1 · 상태: 내부 검토용 초안
근거: `context/gyeonggi-2027-ai-livestock-platform-context.md`

## 1. 한 줄 포지셔닝
**"경기도 102.3억 AI 축산행정 플랫폼의 데이터 입력단(SmaXtec 센서)과 분석단(CowTalk AI)을
동시에 책임지는 유일한 현장형 파트너."**

## 2. 사업 개요 (사실)
- 발주 경기도, 정책연구 Antller Inc.(최종보고 v1.1.0). 규모 5개년 총 102.3억원.
- 사후 대응 → 예측·예방·정밀 행정 전환. 출발 = 낙농 ICT 시범사업(31시군×5농장×50두=7,750두, RCT).
- 보고서가 SmaXtec 반추위 센서 명시 + "실시간성 한계 → 본사 기술 제휴 필수" 적시.

## 3. 우리의 2대 진입축
1. SmaXtec 센서 공급 — 하현제/D2O = 국내 SmaXtec 위내센서 공급자. 7,750두 공급·설치·운영·구독. (smaxtec-sensor-operations)
2. CowTalk AI 탑재 — 플랫폼의 AI·의사결정 레이어. 센서 → 번식·질병·분만 예측·경보 → 행정 대시보드. (cowtalk-ai + eco-bit-platform)
→ "센서도 우리, 두뇌도 우리." 단일 책임·빠른 실증·낮은 통합 리스크.

## 4. 왜 우리인가
1. 대체 불가 공급 채널 — 보고서가 지목한 SmaXtec의 국내 공급/제휴선이 우리.
2. 현장 실증 능력 — 제네틱스·고려동물병원·송영신목장 기반 수의·번식·운영 역량으로 7,750두 RCT 실행 가능.
3. 두뇌 보유 — CowTalk AI로 "수집→예측→행정 액션" 폐루프 제공.

## 5. 102.3억 6개 대과제 중 우리 타깃
- 시범운영·고도화·확산 47.8억 ★핵심 / AI 분석엔진 11.5억 ★CowTalk 탑재 / 데이터통합 플랫폼 19.0억 △Eco-BIT 연계 /
  운영·보안·업무지원·감리 28.1억 ○부분. → 1차 집중 ≈ 59.3억(시범+AI엔진).

## 6. 협력 구조(안)
경기도(발주) ─ Antller(정책연구/주관 후보) ─ D2O/하현제 ─ ①SmaXtec 본사(공급·제휴) + ②CowTalk AI + Eco-BIT(연동).
D2O = 센서 공급 + 현장 실증 + AI 분석 통합 수행 파트너(컨소시엄 형태).

## 7. 리스크 & 대응
- SmaXtec 제휴 미확정 → 국내 독점·우선공급 LOI 선확보.
- 센서 실시간성 한계 → 배치+엣지 수집·Eco-BIT 버퍼링.
- 범용 생성형AI 중복 우려 → 축산 특화 정형데이터·예측으로 차별화.
- 데이터 권한(혈통·유우검정) → 농협·종축개량협회 협조 별도 트랙.

## 8. 액션(30/60/90일)
- 30일: SmaXtec 본사 국내 공급/제휴 LOI 요청 + 본 메모를 경기도/주관용 2~3p 제안 요지로 확장.
- 60일: 7,750두 시범사업 CowTalk+SmaXtec 실증 설계서 작성.
- 90일: 컨소시엄 역할분담 합의 + 59.3억 영역 수행 견적 초안.
