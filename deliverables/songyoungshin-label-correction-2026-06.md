## 4. 라벨 표시사항 정정 체크리스트 (공통)

축산물 표시기준상 라벨에는 영양성분 외 아래 항목이 함께 정정·검증되어야 합니다.

- [ ] **제품명 / 식품유형** — 위 3절대로 확정
- [ ] **영양성분표** — 1·2절 정정값으로 교체 (특히 우유: 열량 78.5 / 지방 4.5 / 포화지방 2.03·14% / 콜레스테롤 13.47)
- [ ] **표시 기준 단위** — 우유 100mL vs 100g, + 포장(총 내용량/1회 제공량) 당 표기 확정
- [ ] **원재료명 및 함량** — 원유(국산, 저지) 100% / 요거트는 유산균 균종 명기
- [ ] **내용량·열량** (예: 200 mL ( ○○ kcal ) )
- [ ] **영업소 명칭·소재지** — 농업회사법인 주식회사 디투오 / 송영신목장
- [ ] **보관방법·유통기한·소비기한** — 냉장 보관
- [ ] **A2 표시** — "A2 베타카제인" 등은 사실 표현만, 질병·치료 효능 표현 금지 (표시광고법)
- [ ] **유산균수**(요거트) — 시험 후 CFU/mL 또는 CFU/g 표기

### 4-2. ⚠️ 반드시 확인 — 자가품질검사 리스크 (우유 유지방 4.5%)
- 품목제조보고 성분규격을 **"유지방 4.5% 이상"** 으로 보고하면, 이후 **자가품질검사에서 4.5% 미만이 나오면 규격 부적합(행정처분 대상)** 입니다.
- 이번 시험 샘플은 **4.12%** 였습니다(6월/하절기 저지방 경향 가능). 연중 4.5% 보장이 어려우면 두 가지 안전안 중 택일을 권고드립니다.
  - **(A) 표준화 보장안(권장)**: 생산공정에서 유지방을 4.5% 이상으로 표준화 → 규격 "4.5% 이상" 유지. 가장 깔끔하고 브랜드("저지 고지방 4.5%") 메시지와도 일치.
  - **(B) 안전 규격안**: 품목제조보고 규격은 **"유지방 4.0% 이상"** 으로 보고하고, 영양성분 표시값만 4.5g로 운영(측정값 4.12 ≤ 표시 4.5는 식약처 지방 허용오차 '표시량의 120% 미만' 충족 → 표시 자체는 적합).
- **추가 권고**: 품목제조보고/공식 라벨 제출용으로는 **유지방 4.5% 표준화 제품을 다시 분석 의뢰**해 실측 성적서로 제출하는 것이 가장 안전합니다(환산값은 잠정).
