# Farm Sensor Installation Checklist Workflow

Use for each farm's smaXtec installation, from pre-visit to confirmed data flow.
Owner: smaXtec Sensor Operations Agent.

## Pre-installation
- [ ] Farm/herd info confirmed (animals, parity, barn layout)
- [ ] Sensors allocated; serial list prepared
- [ ] Installer & veterinarian scheduled
- [ ] Base station / connectivity checked
- [ ] Farmer briefed; consent/contract in place

## Installation
- [ ] Bolus administered per animal (record animal ↔ sensor serial)
- [ ] Base station installed & online
- [ ] Animals registered in system

## Post-installation registration
- [ ] Sensor activation confirmed
- [ ] Data reception verified (each animal)
- [ ] Farm appears on dashboard
- [ ] Baseline period started

## Handover
- [ ] Farmer training delivered (use farmer-sensor-training-material)
- [ ] Subscription/billing set up
- [ ] AS contact & process explained

## Output
Completed per-farm checklist + animal↔sensor mapping + activation/data-reception
confirmation + handover record.

## Rule
No animal goes unmapped; verify data reception before sign-off. Use mock data if API
not yet connected.
