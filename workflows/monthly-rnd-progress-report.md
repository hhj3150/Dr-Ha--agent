# Monthly R&D Progress Report Workflow

Use each month to summarize our participating-company R&D progress. Owner:
Government R&D Secretary Agent. Output uses monthly-progress-report-template.

## Inputs
- Planned tasks for the month
- Actual work done (from team / lab / technical agents)
- Data / requests from 주관기관
- Budget spent this month by category
- Performance events (patent filing, paper, prototype, test result)

## Steps
1. Compare planned vs actual tasks.
2. Note deliverables and evidence (연구노트 references).
3. Update performance indicators.
4. Record budget execution (인건비/재료비/외주비/장비비).
5. List issues, risks, and next-month plan.
6. List documents to submit to 주관기관.

## Output
A completed monthly progress report + updated indicator tracker + action list with owners.

## Rule
Accurate, evidence-backed; mark unconfirmed numbers as draft. Pull technical detail
from the relevant technical agent (CowTalk / D2O / Genetics).
