# 경기도 낙농 ICT 시범사업 — D2O 실증 파트너 제안서 (v2 · V1.1.0 기준)

수신: 경기도 / 주관기관(Antiller 등) · 제안: 농업회사법인 ㈜D2O · 작성: 하실장
분류: Tier 1 / Phase 1 · 상태: 내부 초안(대외 제출 전 검토 필요)
근거: `context/gyeonggi-2027-ai-livestock-platform-context.md` (최종보고 V1.1.0)
> 갱신(2026-06-20): 센서 규모 V1.1.0(파일럿 750 + 본확산 6,750 = 7,500두) 반영. 구 v1(7,750두) 폐기.

## 1. 제안 한 줄
**"낙농 ICT 시범사업(파일럿 750두 → 본확산 6,750두 = 7,500두)에, SmaXtec 위내센서(입력) + CowTalk AI(분석)
+ Eco-BIT(행정 연동)를 일괄 제공하여, 사후 대응 행정을 예측·예방 행정으로 전환하는 실증을 D2O가 책임 수행한다."**

## 2. 시범사업 구조 (V1.1.0 기준)
- 규모: **파일럿 750두(2~3개 시군) → 본확산 6,750두(31개 시군) = 총 7,500두** (농장당 착유우 50두 기준 ≈ 150농장)
- 기간: 3단계 36개월 — ①ICT 도입·기반 0~6M ②AI 플랫폼·데이터 통합 7~24M ③정책 실증·확산 25~36M
- 설계: 선정·비선정 농가 데이터 모두 확보 → **무작위비교시험(RCT)**. 파일럿이 RCT 실증의 핵심 코어.

## 3. D2O 적용 설계 (3-레이어)
### ① 센서 레이어 — SmaXtec (smaxtec-sensor-operations)
- 위내(볼러스) 센서 공급·설치·등록·활성화·데이터 수신 확인·AS·구독.
- 측정: 반추·활동·체온·pH·음수 → 소모성 질병(유방염·케토시스·유열·제엽염) 사전징후.
- 설치 표준: 농장당 50두, 베이스스테이션·연결성 점검, 개체↔센서 매핑(템플릿: sensor-installation-checklist).

### ② AI 레이어 — CowTalk AI (cowtalk-ai)
- 센서·농장·공공데이터 통합 → 번식(발정·수정적기·임신), 질병위험, 분만예측, herd 이상탐지.
- 모든 경보는 6대 질문으로 출력: 무엇이/왜/조치/언제/누구에게/대시보드 표현.
- 사용자별: 농가(간단 조치)·수의사(임상)·수정사(적기)·행정(지역 모니터링·위험분석).

### ③ 연동 레이어 — Eco-BIT (eco-bit-platform)
- 센서·CowTalk 인사이트를 표준화·메타데이터 정의 → 경기도 행정 대시보드/RAG·MCP에 연결.
- 농가-행정 데이터 경로 분리, 행정용은 집계·익명화. (정책연구의 표준화·RAG·MCP 방향과 정합)

## 4. RCT 실증 설계
- 처치군(센서+CowTalk 도입 농가) vs 대조군(비선정 농가) 비교.
- 측정지표(예시·파일럿 목표): 소모성 질병 **조기검출률↑/발병률↓**, 발정 발견율·**수태율↑**, 분만사고↓,
  행정 인지~대응 **소요시간↓**, 농가 노동·비용↓, 데이터 완결성↑.
- 데이터 수집 시트·임신/분만 추적: pregnancy-result-tracker / calving-result-tracker 템플릿 활용.

## 5. 3단계 일정 & 마일스톤
| 단계 | 기간 | D2O 핵심 활동 | 산출 |
|---|---|---|---|
| 1 도입·기반 | 0~6M | 파일럿 750두 설치·데이터 수신 검증, 농가 교육, 본확산 물량 준비 | 파일럿 설치완료·수신율 KPI |
| 2 AI·통합 | 7~24M | CowTalk 경보 운영, Eco-BIT 행정 대시보드 연동, 본확산 6,750두 단계 설치, RCT 데이터 축적 | 예측·경보 운영·중간 성과 |
| 3 실증·확산 | 25~36M | 정책효과 실증(RCT 분석), 한우·양돈 확산 설계 | 실증 레퍼런스·확산 가이드 |

## 6. 역할 분담(안)
- D2O: 센서 공급·설치·운영 + CowTalk AI 분석 + 현장 실증 총괄 수행.
- SmaXtec 본사: 센서 공급·기술 제휴(API·실시간성 보완 협의).
- Eco-BIT: 데이터 표준화·연동·대시보드.
- 경기도/Antiller: 농가 선정·행정 연계·정책 실증 프레임.
- 수의/현장(고려동물병원·제네틱스): 번식·질병 검증, RCT 임상 지원.

## 7. 성과지표(KPI) — 파일럿 목표(실측 전 추정)
질병 조기검출률 · 발병률 감소 · 수태율 개선 · 분만사고 감소 · 행정 대응시간 단축 · 데이터 수신율 ·
농가 만족도. (수치는 실증 데이터 확정 전까지 목표치로 표기)

## 8. 예산 연계 (189.2억 중)
- 본 실증 = "시범운영·고도화·확산 78.2억"(파일럿 13.4 + 본확산 64.0 + 표준화 2.0) + "AI 분석·예측 엔진 22.4억" 영역과
  직접 정합(≈100.6억 권역).
- D2O 수행 견적은 센서 수량(7,500두)·구독·설치·AI 운영 기준으로 별도 산정(3·4번 문서).

## 9. 리스크 & 대응
