## ✅ 정정·철회 정리 (검토자 과오 포함)

### 1. 우유 살균조건 — ✅ 정정: 불일치 아님 (검토자 오독)
- 재확인 결과 750ml·180ml **둘 다 "HTST 살균(75℃, 20초)"로 동일**.
  (※ v1 최초본에서 750ml을 "30초"로 적은 것은 저배율 판독 오류 — 정정함.)
- `20초` ≥ HTST 법정기준(72~75℃ / 15초) 충족 → **적합. 수정 불필요.**

### 2. 트랜스지방 — 함량 0g은 정확. "0%" 한 칸만 경미한 형식 차이
- **트랜스지방 `0 g`(함량)은 정확** — 타사도 동일. **문제 아님.**
- 유일한 차이: 옆 **`0 %`** 한 칸. 트랜스지방은 1일 영양성분 기준치가 없어
  **표준 서식에선 % 칸을 비우고 'g'만** 표기(타사 라벨이 그러함). 우리 라벨은 "0g"에 "0%"를 덧붙인 것.
- **중요도: 매우 경미. 강제 수정 아님.** 다음 인쇄 때 % 칸만 비우면 더 정석 — **그대로 둬도 실질 위험 낮음.**
  (v1에서 🔴 즉시로 분류한 것은 과함 → 하향 정정.)

## 🟠 검증 필수 — 요거트 영양분석값 계통오류 의심

### 3. 요거트 포화지방 ≈ 총지방 (≈100%) — 유지방 특성상 불가능
| | 총지방 | 포화지방 | 포화/총 |
|---|--:|--:|--:|
| 우유 750ml(100mL당) | 4.5g | 2.5g | **56%** ✅ 정상 |
| 요거트 500ml(100mL당) | 4.9g | **4.8g** | **98%** ⚠ |
| 요거트 180ml(180mL당) | 8.6g | **8.6g** | **100%** ⚠ |
- 유지방의 포화지방 비율은 통상 **약 60~65%**. 같은 A2 원유로 만든 요거트가 ~100%일 수는 없음.
- 두 요거트 라벨이 **동일하게** 나옴 → 단순 오타가 아니라 **분석성적서/입력 계통오류**로 판단.
- 조치: 요거트 **영양성분 분석성적서의 포화지방 재확인**(예상 정상값 ≈ 3.0g/100mL 내외).

### 4. 요거트 콜레스테롤 과다 (우유의 약 3배)
- 우유 ≈ 14~15mg/100mL vs 요거트 ≈ **42mg/100mL**(500ml 42 / 180ml 75 환산 41.7).
- 지방함량이 비슷한데 콜레스테롤만 3배 → 위 3번과 함께 **재분석 권장**.

## 🟡 권고 / 일관성 정비

### 5. 영양표시 기준단위 — ✅ 철회(현행이 맞음)
- 750ml·500ml = "100mL 당" / 180ml 2종 = "180mL(총 내용량) 당".
- **현행이 정석.** 180mL 같은 소용량(1회 섭취 분량)은 **총 내용량당 표기가 올바른 방식**이며,
  과거 하실장 안내도 그러했음. **"100mL로 통일" 권고는 자기모순 → 철회.** 현행 유지.

### 6. 우유 100mL당 열량 미세 불일치 (참고)
- 750ml 76kcal/100mL vs 180ml 75kcal/100mL(환산). 반올림 차이 수준 — 굳이 손댈 필요 낮음(참고만).

### 7. 발효유 "유형" — ✅ 철회(발효유 등록 적법, 회장님 결정)
- 무지유고형분 8.40%는 **발효유 기준(무지유고형분 3.0% 이상)을 충족** → **"발효유"로 등록·표시 적법.**
- 농후발효유(8% 이상)는 별도 '선택 가능' 유형일 뿐 **강제 아님**. **회장님 결정: 발효유로 등록 → 그대로 적합.**

### 8. 제조원 일치 확인
- 제조원 **"농업회사법인 주식회사 디투오 / 경기도 안성시 미양면 미양로 466"**.
- 브랜드(송영신목장)와 제조원이 다른 구조는 가능하나, **축산물가공업 허가·품목제조보고 주체와 동일한지** 확인.

## 🔵 「식품표시광고법」 별도 점검 (표시기준과 별개, 실증자료 필요)
전면 강조·광고 문구는 표시기준이 아닌 **표시·광고 실증제** 대상 — 근거자료 구비 필수:
- **"100% A2저지"** → 전 사육우 **A2A2 유전형 검사 성적**으로 입증 가능해야.
- **"대한민국 단 0.3% 희귀 저지"** → 0.3% 산출 **출처·근거** 보관.
- **"친환경(유)"** → 친환경축산물 인증서, **"동물복지 인증목장"** → 동물복지축산농장 인증서(마크 표시 확인됨).
