## PART A — SmaXtec 본사 LOI 요청 (국내 공급·기술제휴 확인)

### A-1. 요청 취지 (한 줄)
경기도 189.2억 AI 축산행정 플랫폼(공공)에 SmaXtec 위내센서 **7,500두**를 투입하기 위해,
하현제의 **국내 독점 공급권·API 연계**를 공공조달 맥락에서 재확인하는 LOI를 본사로부터 확보.

### A-2. 핵심 확인 항목 (LOI에 담을 조항)
1. 하현제(D2O)의 **대한민국 내 SmaXtec 센서 독점 공급·판매권** 확인 (기확보 권리의 공공사업 적용 명문화).
2. **SmaXtec Cloud API 국내 연계권** 확인 — CowTalk AI 분석 결합 및 Eco-BIT 연동 목적.
3. 보고서가 지적한 **대량 동시호출 실시간성 한계 보완**을 위한 본사 기술협력 의향
   (배치 최적화·호출한도 상향·엣지 수집 협의 창구).
4. 공공 5개년 사업 기간 **공급 연속성·무상교체(고장)·AS 정책** 유지 확인.
5. 유효기간·비독점 정보보호(NDA 연계) 조항.

### A-3. 발송 형식
- 1차: 영문 LOI Draft(본 항목 1~5 기반) + 국문 요지 1p 동봉.
- 수신: SmaXtec HQ(오스트리아) 파트너십 담당 + 코리아제네틱스(수입사) 참조.
- 회신 목표: **30일 내 서명 LOI 또는 의향 회신.**

> LOI는 "신규 권리 요청"이 아니라 **기확보 권리의 공공사업 적용 확인**임을 명확히 하여 협상 마찰 최소화.

## PART B — 코리아제네틱스 공공사업 물량·단가 합의 요청

### B-1. 요청 취지
시범~본확산 **7,500두(파일럿 750 + 본확산 6,750)** 의 안정적 공급을 위해,
수입사 코리아제네틱스와 **공공사업 전용 물량·수입단가·납기**를 사전 합의한다.

### B-2. 합의 항목
| 항목 | 합의 목표 |
|---|---|
| 물량 | 7,500두(2단계: 750 선행 → 6,750 확산) + 무상교체 예비분 |
| 수입단가 | 공공사업 전용 단가(현 기준선 대비 물량 할인 협의) |
| 납기·재고 | 파일럿 0~6M 우선 납품 + 본확산 단계별 분할 납품, 재고 버퍼 확보 |
| 환율·일정 | 환율 변동·수입 일정 리스크 분담 방식 |
| 무상교체 | 4년 유지조건 고장 무상교체 물량의 공급책임 명확화 |

### B-3. 확정 시 효과
- gyeonggi-03 단위경제의 **원가·설치비 가정치 → 실수치 확정** → gyeonggi-04 견적 v1 산출 가능.
- 경기도/주관 제출 견적의 **공급 안정성 근거**(물류·재고·납기) 확보.
