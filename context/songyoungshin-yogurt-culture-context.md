## 2. 용어 / 기본 특성

- **YO-MIX®**: Danisco의 **호열성(thermophilic) 요거트 스타터 컬처** 브랜드. 기본 균은
  *Streptococcus thermophilus* + *Lactobacillus delbrueckii* subsp. *bulgaricus* 조합.
  발효온도대 약 **40~43°C** (전형적 요거트 발효), pH 약 4.5~4.6 도달 시 종료.
- **LYO**: Lyophilized = 동결건조 분말. **DVS/DVI(직접 접종)** 방식 — 모균 배양 없이
  원유에 직접 투입.
- **DCU**: DuPont Culture Unit = 접종량(역가) 단위. 숫자가 클수록 더 많은 우유를 발효.
  - T13 = **200 DCU/봉** (대용량), 300 = **50 DCU/봉** (소량). → T13가 주 발효 주력,
    300은 소량 보조·풍미/밸런스 조정용으로 운용하는 구성으로 해석.
- **보관**: ≤4°C 냉장 권장(단기). 장기 보관 시 냉동(−18°C 이하)이 일반적이나 **개봉 후 즉시 사용**.
  개봉·해동 반복 금지(역가 손실).

## 3. 균주별 캐릭터(제품 라인 일반 특성 — 사양서로 최종 확인)

- **YO-MIX T13 (T 시리즈 = Texture 라인)**: 점도·바디감이 강한 **고텍스처/크리미** 계열.
  EPS(균체외다당) 생성으로 **되직하고 부드러운 질감**, 유청분리(syneresis) 억제에 유리.
  세트/스터드 요거트 모두 적합. 산도는 비교적 **마일드(부드러운 신맛)** 지향.
- **YO-MIX 300 (300 시리즈 = 클래식 밸런스 라인)**: **균형 잡힌 산생성과 표준 발효 프로파일**.
  전통적 요거트 풍미·향미 형성에 기여. 텍스처는 중간 수준.
- **블렌드 의도(해석)**: T13의 **풍부한 텍스처/크리미함** + 300의 **클래식한 발효 풍미/밸런스** →
  저지 A2 원유의 높은 유지방·고형분과 결합해 **진하고 부드러운 프리미엄 플레인요거트** 지향.
  (저지유는 유지방·유단백·총고형분이 높아 텍스처 종균과 시너지가 크다.)

## 4. 운영 메모 (Product Development Agent)

- **A2 강조**: 송영신목장 저지는 A2A2 원유 → "A2 저지 플레인요거트"의 핵심 차별점.
  종균은 발효균(텍스처/풍미)이고, **A2/A1은 원유(소)의 베타카제인 유전형**에서 결정됨.
  마케팅 표기 시 "A2 원유 + 프랑스 Danisco 종균" 구분해서 정확히 기술.
- **접종량·발효시간·종료 pH**는 라벨이 아닌 **Product Description + 실제 양산 SOP**에서 확정.
  현재 확정 미보유 → 향후 표준화 SOP 작성 시 채움.
- **재고/로트 추적**: 위 Lot·생산일·유통기한 기록. 동결건조 종균은 로트별 역가 편차 가능 →
  로트 교체 시 1차 시험발효 권장.
