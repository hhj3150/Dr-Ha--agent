## 3. LOT 채번 규칙

**형식**: `제품코드 - 제조일(YYMMDD) - 설비 - 배치순번`

```
M K - 2 6 0 6 1 6 - L 1 - 0 1
└┬┘   └──┬────┘   └┬┘  └┬┘
제품   제조일      설비  배치
```

**예시 (2026.06.16)**
| 제품 | LOT 예시 |
|---|---|
| 우유 | `MK-260616-L1-01` |
| 아이스크림 믹스 | `IC-260616-L1-01` |
| 요거트 | `YG-260616-T1-01` |
| 카이막 | `KM-260616-L2-01` |

## 4. LOT를 새로 끊는 기준 (설비별)

| 설비 | 1 LOT 기준 | 분리 시점 |
|---|---|---|
| **T1 요거트 발효탱크** | **발효 1배치 = 1 LOT** (가장 명확) | 탱크 새로 발효 돌릴 때마다 |
| **L1 우유** | 1일 1회 냉각·충전 배치 = 1 LOT | **원유 로트 변경 시 분리** |
| **L1 아이스크림 믹스** | 1 믹스 배치 = 1 LOT | 우유와 **시간 분리 + 전환 세척 후** |
| **L2 카이막** | 1 가열·응고 배치 = 1 LOT | 배치마다 |

---

## 5. ★공용라인(L1) 핵심 주의 — 우유 ↔ 아이스크림 믹스

같은 냉각라인을 공유하므로 전환 시 **반드시**:
1. **CIP 세척·소독 실시 및 기록** (전환 사이)
2. **잔류(이전 제품) 확인**
3. **LOT 명확 분리** (우유 LOT 종료 → 세척 → 아이스크림 LOT 시작)
4. 아이스크림 믹스에 **가당·부재료(알레르겐)** 투입 시 교차오염 관리

> 📌 공용라인은 점검·리콜 때 가장 먼저 보는 지점입니다. "세척기록 없는 전환 = 교차오염 의심".

## 6. LOT 연결 보관 (추적관리 완성)

하나의 LOT에 다음을 **묶어** 보관 (점검·리콜 시 즉시 추적):

```
원유 수불기록 → 제조일지(CCP) → CIP 세척기록(L1 전환 시) → 자가품질검사 성적서 → 출고·거래처 기록
```

## 8. LOT 표시 위치 (어디에 찍나)

**① 제품 포장(소비자 표시) — 법정**: 최소판매단위 포장의 정보표시면에 **유통기한·제조연월일과 함께** 표시
| 제품 | 위치(예) | 방법 |
|---|---|---|
| 우유(병/팩) | 뚜껑 상단·병 어깨 / 팩 상단 | 잉크젯 자동 인자 |
| 요거트(컵) | 뚜껑 윗면·컵 가장자리 | 잉크젯·각인 |
| 카이막·아이스크림 | 라벨 하단·용기 바닥 | 잉크젯·스티커 |
> 충전라인 끝단에 잉크젯/레이저로 제조일+유통기한+LOT 자동 인자 권장

**② 외포장(박스/케이스)**: 케이스 라벨에 LOT 기재 (B2B·물류 추적)

**③ 내부 서류·전표**: 제조일지 → 자가품질검사 성적서 → 출고전표·거래명세서까지 **동일 LOT** 기재 (리콜 시 거래처별 추적)

> 📌 법적 최소: 정보표시면에 제조연월일 또는 제조번호(LOT) + 유통기한. 1일 다배치면 LOT 인자 필수.
