## 🔴 수정 필수 (Critical)

### 1. 우유 살균조건 불일치 — 같은 제품인데 시간이 다름 ✅ 해결방향 확정
- 750ml: **"HTST 살균(75℃, 30초)"** / 180ml: **"HTST 살균(75℃, 20초)"**
- 두 라벨은 **동일 제품·동일 품목보고번호(…181)** → 살균조건이 달라선 안 됨. 둘 중 하나가 오류.
- **회장님 확인(2026-06): 실제 공정 = 75℃ / 20~30초.**
- 확정 문구(우유 2종 동일): **"HTST 살균(75℃, 20~30초) / 살균제품"**
  - 하한 20초 ≥ HTST 법정기준(72~75℃/15초) 충족 · 실제 공정 반영 · "이상" 등 모호표현 배제.
- 잔여 조치: **품목제조보고서 신고 살균조건과 라벨 문구 일치** 최종 확인.

### 2. 트랜스지방 "0%" 표기 — 4종 전부 (영양표시 서식 오류)
- 현재 모든 라벨이 **"트랜스지방 0g 0%"**로 표기.
- 트랜스지방은 「식품등의 표시기준」 별표5의 **1일 영양성분 기준치가 없는 항목** → **% 칸은 공란**이어야 함. "0%"는 부적합.
- 조치: 4종 모두 트랜스지방의 **% 값(0%) 삭제**(공란 처리).

## 🟠 검증 필수 — 요거트 영양분석값 계통오류 의심

### 3. 요거트 포화지방 ≈ 총지방 (≈100%) — 유지방 특성상 불가능
| | 총지방 | 포화지방 | 포화/총 |
|---|--:|--:|--:|
| 우유 750ml(100mL당) | 4.5g | 2.5g | **56%** ✅ 정상 |
| 요거트 500ml(100mL당) | 4.9g | **4.8g** | **98%** ⚠ |
| 요거트 180ml(180mL당) | 8.6g | **8.6g** | **100%** ⚠ |
- 유지방의 포화지방 비율은 통상 **약 60~65%**. 같은 A2 원유로 만든 요거트가 ~100%일 수는 없음.
- 두 요거트 라벨이 **동일하게** 나옴 → 단순 오타가 아니라 **분석성적서/입력 계통오류**로 판단.
- 조치: 요거트 **영양성분 분석성적서의 포화지방 재확인**(예상 정상값 ≈ 3.0g/100mL 내외).

### 4. 요거트 콜레스테롤 과다 (우유의 약 3배)
- 우유 ≈ 14~15mg/100mL vs 요거트 ≈ **42mg/100mL**(500ml 42 / 180ml 75 환산 41.7).
- 지방함량이 비슷한데 콜레스테롤만 3배 → 위 3번과 함께 **재분석 권장**.

## 🟡 권고 / 일관성 정비

### 5. 영양표시 기준단위 불일치
- 750ml·500ml = **"100mL 당"** / 180ml 2종 = **"180mL(총 내용량) 당"**.
- 법적으론 각각 허용(소용량은 총내용량당 가능)이나, **제품군 통일** 권고(소비자 비교 편의·브랜드 일관성).

### 6. 우유 100mL당 열량 미세 불일치
- 750ml 76kcal/100mL vs 180ml 75kcal/100mL(환산). 동일제품 → **한 값으로 통일**.

### 7. 발효유 "유형" 재확인 — 농후발효유 해당 가능
- 무지유고형분 **8.40%** 표기 → 「가공기준」상 **농후발효유(무지유고형분 8% 이상)** 기준 충족.
- "발효유"로 신고했는지 품목제조보고서와 **유형 일치 확인**. (농후발효유 표기가 프리미엄 포지셔닝에도 유리)

### 8. 제조원 일치 확인
- 제조원 **"농업회사법인 주식회사 디투오 / 경기도 안성시 미양면 미양로 466"**.
- 브랜드(송영신목장)와 제조원이 다른 구조는 가능하나, **축산물가공업 허가·품목제조보고 주체와 동일한지** 확인.
