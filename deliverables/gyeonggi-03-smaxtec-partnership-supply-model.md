# SmaXtec 본사 기술제휴 + 국내 공급/단가 모델 (v1)

작성: 하실장 · 담당: smaxtec-sensor-operations + ceo-strategy · 분류: Tier 1 / Phase 1
상태: 내부 초안(수치는 협의 전 가정치 — 본사·경기도 확정 필요)
근거: `context/gyeonggi-2027-ai-livestock-platform-context.md`, `context/cowtalk-business-context.md`

## 1. 목적
경기도 정책연구가 명시한 "SmaXtec 반추위 센서 + 본사 기술 제휴 필수"의 그 주체를
**하현제/D2O = 국내 공급·제휴선**으로 공식화하고, 시범사업 7,750두 공급의 단가·구조를 설계한다.

## 2. 제휴 논리 (왜 D2O가 국내 채널인가)
- 경기도 102.3억 플랫폼의 **공식 센서 수요처**가 확정 단계 → 본사 입장에서 **대량·정부 레퍼런스 채널**.
- D2O는 센서만 파는 벤더가 아니라 **CowTalk AI(분석)+현장 실증+수의 역량**을 묶어 **데이터 가치를 끌어올리는 파트너** → 본사 데이터 활용도·갱신율↑.
- 따라서 단순 리셀러가 아닌 **국내 독점/우선 공급 + 기술 제휴(API·실시간성)** 를 요구할 명분이 있다.

## 3. 제휴 요청 항목 (본사에 요구할 LOI 골자)
1. **국내(또는 경기도 공공사업) 우선·독점 공급권**
2. **API 기술 제휴** — 대량 호출 실시간성 한계 보완(호출한도 상향, 웹훅/스트리밍, 엣지 캐싱 협의)
3. **공공사업 특화 단가** — 7,750두 규모 볼륨 디스카운트 + 5개년 구독 구조
4. **데이터 사용 합의** — CowTalk AI 학습/분석 목적의 데이터 활용 권한·범위
5. **AS/교체·재고** — 국내 재고·설치기사 운영 지원, 고장률 대응 SLA

## 4. 공급 단가 모델 (구조 — 금액은 가정치, 확정 전)
> 실제 단가는 본사 협의로 확정. 아래는 구조와 변수 정의(❏=확정 필요).

| 항목 | 단위 | 비고 |
|---|---|---|
| 볼러스 센서 | 두당 1회 | 하드웨어 공급가 ❏ |
| 베이스스테이션/게이트웨이 | 농장당 | 농장 규모별 ❏ |
| 설치·등록·교육 | 농장당/두당 | 설치기사 인건 포함 ❏ |
| 월 구독료(데이터·CowTalk 분석) | 두당/월 | 핵심 반복매출 ❏ |
| AS·교체 | 연 % | 고장률 가정 ❏ |

- 매출 = (하드웨어 7,750두 공급) + (설치 155농장) + (월 구독 두당×7,750×개월) + (AS).
- 핵심: **월 구독(두당) = 5개년 반복매출 엔진** → CowTalk AI 분석 가치가 구독 단가를 정당화.

## 5. 7,750두 공급·설치 운영 계획
- 농장: 31시군 × 5농장 = 155농장, 농장당 50두.
- 단계: 1단계(0~6M) 설치 집중 → 데이터 수신율 검증 → 2단계부터 구독 과금.
- 운영도구: sensor-installation-checklist · monthly-sensor-subscription-report · sensor-as-response-template.
- 인력: 권역별 설치기사 배정 + 수의 현장점검(고려동물병원·제네틱스 연계).

## 6. 리스크 & 대응
- 본사 제휴 지연 → LOI 우선 확보로 입찰 명분, 병행 대체 협의 트랙.
- 실시간성 한계 → 배치+엣지 수집, Eco-BIT 버퍼링, 본사 API 협의.
- 환율·수입 일정 → 재고 버퍼·물류 일정 관리(공급계획표).

## 7. 액션

- 60일: 7,750두 기준 **공급·구독 견적 v1** 산출(가정치→확정치 치환).
- 90일: 공공사업 특화 계약·SLA 합의.
