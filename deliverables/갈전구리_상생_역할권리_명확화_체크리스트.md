# 갈전구리 — 상생 기반 역할·권리 명확화 체크리스트 (v1)

> 작성: 하실장 · 2026-06-13 · 상태: 협의용 초안(Draft)
> 법적 조항은 **변리사·변호사 검토 필수**(아래 ⚖️ 표시 항목).
> 대원칙: **㈜지엠더블유·충남대와의 상생.** 명확성은 다툼을 막는 상생의 장치이지 독식이 아니다.

## 0. 대원칙 — 각자 강점에서 정당하게 (Win-Win)
| 파트너 | 강점·역할 | 정당한 수익 |
|---|---|---|
| **㈜지엠더블유** | 로봇 설계·제조·하드웨어 | 로봇 제조·납품 마진 |
| **충남대 산학협력단** | 자율주행 원천기술(전용실시권) | **경상기술료 매출 5%** |
| **㈜디투오** | 사업화·판매·솔루션·피트모스 소모품 | 솔루션·소모품·서비스 수익 |
> 목표는 "누가 더 갖나"가 아니라 **"각자 강점으로 함께 키워 사업을 성공"**시키는 것.

## A. 계약·권리 명확화 ⚖️ (합의로 명문화)
- [ ] **전용실시권 범위** — 충남대 특허(10-2145902, 10-2340543)의 전용실시 범위·지역·기간 명시.
- [ ] **완성품 판매권 귀속** — 갈전구리 **판매·유통은 D2O(사업화 주체)** 가 담당함을 명문화. *GMW=제조, D2O=판매·솔루션* 경계를 합의로 분명히.
- [ ] **경상기술료 5% 적용범위** — "기술적용 제품 총매출"의 정의(로봇 본체만? 소모품·서비스 포함 여부?)를 명확화 → 소모품·서비스까지 5%가 붙지 않도록 합리적 경계 합의.
- [ ] **개량발명(Improvement) 귀속** — 사업화 과정에서 D2O가 만든 개량·응용 발명의 소유·실시 권리 사전 합의.
- [ ] **공동소유 IP 지분·실시조건** — 공동개발 결과물의 지분율·각자 실시 자유 명시.
- [ ] **독점/비독점·우선권** — D2O 사업화 영역의 독점성 또는 우선협상권 수준 합의.

## B. 상표·디자인 (상생 동의 위에서 선점)
- [ ] **'갈전구리' 상표 출원** — 사업화 주체인 **D2O 명의로 출원**하되, **컨소시엄 사전 동의·역할 확인** 후 진행(공동성과 존중).
  - 지정상품류(예): **7류(농기계)**, 9류(제어 S/W), 35류(판매업), 31류(축산 관련) 등 ⚖️ 변리사 협의.
  - 국문 '갈전구리' + 영문 표기 동시 확보.
- [ ] **디자인권** — 로봇 외형 디자인권(주체·공동 여부 합의).
- [ ] **도메인·SNS 핸들** 선점.

## C. 소모품·데이터 (D2O 영역, 호환은 열어둠)
- [ ] **피트모스 최적화** — 로봇이 D2O 피트모스에 최적 동작하도록 설계 사양 반영(타 깔짚도 동작은 가능하게 — 농가 선택권·상생).
- [ ] **운영·교반 데이터 소유권** — 실증·운영 데이터의 소유·활용 권리 합의(공동 연구활용은 허용).

## D. 상생 장치 (분쟁 예방·성과 공유)
- [ ] **정기 협의체** — 3자 분기 운영회의(역할·매출·이슈 공유).
- [ ] **공동 레퍼런스·공동 홍보** — 실증 성과를 3자 공동 명의로 발표(각자 신뢰도↑).
- [ ] **분쟁 예방 조항** — 권리 다툼 시 협의·조정 절차 사전 명문화.
- [ ] **성과 확산 시 재투자/배분 원칙** — 사업 확대 이익의 공정 배분 방향 합의.
