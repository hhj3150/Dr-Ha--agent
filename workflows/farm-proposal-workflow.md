# Farm Proposal Workflow (Peat Moss)

Use to create a per-farm peat-moss proposal that closes a sale. Owner: D2O Peat
Moss Commercial Agent (tech effect from D2O Environment Carbon Agent).

## Inputs
Species, herd size, barn type, current bedding (e.g., sawdust), pain points (odor,
moisture, hoof/udder health, labor, manure handling).

## Steps
1. Diagnose current situation & cost.
2. Recommend product & usage scenario (depth, refresh cycle, Dr. Ha Bedding / HBP).
3. Quantify expected benefits (odor↓, welfare↑, compost value↑, labor↓) as expected/pilot.
4. Build ROI: bedding cost vs savings + premium-compost upside.
5. Pricing & quote (draft), delivery & MOQ.
6. Trial offer + next steps.

## Output
A 1–2 page farm proposal (use peatmoss-farm-proposal-template) + quote (use
peatmoss-price-quote-template) + ROI table.

## Rule
Describe Dr. Ha Liquid by function only. Mark effects as expected/pilot and prices as
draft until confirmed. Recommend a short on-farm trial.
